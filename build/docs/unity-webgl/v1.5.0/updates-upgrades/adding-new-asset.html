Adding New Asset

Provide the following asset details.

  1. Update & Add new assets in your Game Assets and Game Asset Components Sheet.
  2. Download Sheet as .csv files upload it here, also similarly upload the reference asset prefabs for the new asset here
  3. You will receive an updated SpacesAssets.cs file and additional Tracker and Loader Script for the new asset and its components.
  4. Follow the same steps mentioned for the Asset & CDN integration.
  5. You can add asset Interactions by using the code snippet provided and following the Asset Integration steps here. The code snippet will be similar to the one below
spacesEngine.AssetUpdateInteraction(SpacesAssets.GetAssetId("your-asset-name"),interactionValue); //pass your interactionValue
